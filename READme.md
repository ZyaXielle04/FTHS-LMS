# Project Overview

This is a Learning Management System built using Firebase Realtime Database (RTDB) that supports three
distinct user roles: Admin, Teacher, and Student. The system provides role-based access control and
functionality tailored to each user type.

# System Features

- Role-based authentication and authorization
- Real-time data synchronization using Firebase RTDB
- User management for administrators
- Course management for teachers
- Learning portal for students
